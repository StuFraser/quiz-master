# quiz-master


A simple web-based quiz application where users can select topics, answer multiple-choice questions, and see their results.

---

## Features

- Select quiz topic, number of questions, and answers per question
- Navigate through questions with Next/Back buttons
- Display images for questions (e.g., country flags)
- Mobile-friendly and responsive layout
- Uses HTML, CSS, and vanilla JavaScript
