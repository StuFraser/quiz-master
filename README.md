# Quiz Master

A configurable single-page application built with vanilla HTML, CSS and JavaScript — no frameworks, no build step.

The UI is rendered dynamically through JavaScript, with content injected into the DOM as the user progresses rather than navigating between pages. Rather than a fixed quiz, Quiz Master lets users tailor their experience before starting — choosing the topic, how many questions to answer, and how many answer options to include. Questions are presented one at a time with back/forward navigation, and results are shown at the end.

## Features

- Select quiz topic, number of questions, and number of answer options per question
- Navigate through questions with Next/Back buttons
- Image support for visual questions — includes a country flags category where a flag is displayed and you identify the country

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript
